# Branding assets

`nucleus-256.png` (256x256) and `nucleus.ico` are generated by `scripts/make-branding.ps1` from the mod
palette (teal `#33C4C4`, magenta `#FF66C2`). They are committed so builds/installer don't depend on a
Windows render step.

## Regenerate

```powershell
pwsh scripts/make-branding.ps1          # re-render both PNG and ICO from the script
```

## Swap in real artwork

Replace `nucleus-256.png` with your own 256x256 PNG (transparent background, square), then rebuild the
multi-size `.ico` from it:

```powershell
pwsh scripts/make-branding.ps1 -IcoOnly  # keep the PNG, regenerate nucleus.ico from it
```

The `.ico` embeds 16/32/48/64/128/256 px images (PNG-compressed, Vista+ format).
